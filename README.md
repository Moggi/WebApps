# WebApps
Apps made to learn the [Django's](https://github.com/django/django) and [Heroku's](https://www.heroku.com) way of creating WebApps

## About
This was made with:
- Docker
- Postgres (container)
- Django (container)
- An Docker self-made **bridge network** to connect the containers
